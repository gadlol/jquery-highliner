jquery-highliner
================

HighLiner displays two panels. One above cursor and one bellow, so you can focus only where your cursor is. Its very usefull if you are reading large texts such as paragraphs. Press Activate Highliner to start the magic!!!. You will find documentation for options, and examples inside the jquery.highliner-0.0.3.js file.
